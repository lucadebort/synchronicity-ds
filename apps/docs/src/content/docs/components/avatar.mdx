# Avatar

Avatar displays a user or entity representation. It supports images, text initials, or icon fallbacks, and can be grouped for showing multiple users.

## Variants

### Icon (Default)

Displays an icon or emoji when no image or initials are provided.

<PreviewBox class="flex-row gap-lg">
  <div class="sync-avatar md"><span class="avatar-icon">☯</span></div>
  <div class="sync-avatar md"><span class="avatar-icon">☰</span></div>
  <div class="sync-avatar md"><span class="avatar-icon">⚋</span></div>
</PreviewBox>

```tsx
<Avatar icon="☯" />
<Avatar icon="☰" />
<Avatar icon="⚋" />
```

### Initials

Displays up to 2 characters when no image is available.

<PreviewBox class="flex-row gap-lg">
  <div class="sync-avatar md initials"><span>JD</span></div>
  <div class="sync-avatar md initials"><span>AB</span></div>
  <div class="sync-avatar md initials"><span>YI</span></div>
</PreviewBox>

```tsx
<Avatar initials="JD" />
<Avatar initials="AB" />
<Avatar initials="YI" />
```

### Image

Displays a user photo or profile image.

<PreviewBox class="flex-row gap-lg">
  <div class="sync-avatar md image"><div class="avatar-image-placeholder g1"></div></div>
  <div class="sync-avatar md image"><div class="avatar-image-placeholder g2"></div></div>
  <div class="sync-avatar md image"><div class="avatar-image-placeholder g3"></div></div>
</PreviewBox>

```tsx
<Avatar source={{ uri: 'https://example.com/photo.jpg' }} />
```

## Custom Background

Override the default background color for initials/icon avatars.

<PreviewBox class="flex-row gap-lg">

  <div class="sync-avatar md initials success-bg"><span>✓</span></div>


## API Reference

### Avatar Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `source` | `ImageSourcePropType` | - | Image source |
| `initials` | `string` | - | Fallback initials (max 2 chars) |
| `icon` | `string` | `'☯'` | Fallback icon/emoji |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Avatar size |
| `backgroundColor` | `string` | - | Custom background color |
| `style` | `ViewStyle` | - | Custom styles |
| `accessibilityLabel` | `string` | - | A11y label |

### AvatarGroup Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | **Required** | Avatar elements |
| `max` | `number` | `4` | Max visible avatars |
| `size` | `AvatarSize` | `'md'` | Size for all avatars |
| `style` | `ViewStyle` | - | Custom styles |

## Complete Example

```tsx
import { Avatar, AvatarGroup, Card, Badge } from '@synchronicity/react-native';
import { View, Text } from 'react-native';

function ReadingParticipants({ participants }) {
  return (
    <Card variant="outlined" padding="md">
      <View style={styles.header}>
        <Text style={styles.title}>Shared Reading</Text>
        <Badge variant="info" size="sm">{participants.length}</Badge>
      </View>

      <AvatarGroup max={4} size="sm">
        {participants.map((p) => (
          <Avatar
            key={p.id}
            source={p.avatar ? { uri: p.avatar } : undefined}
            initials={p.initials}
            accessibilityLabel={p.name}
          />
        ))}
      </AvatarGroup>

      <Text style={styles.names}>
        {participants.map(p => p.name).join(', ')}
      </Text>
    </Card>
  );
}
```
