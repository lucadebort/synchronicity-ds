## Basic Usage

<PreviewBox class="p-lg">
  <div class="modal-demo">
    <div class="modal-backdrop"></div>
    <div class="modal-content">
      <div class="modal-header">
        <span class="modal-title">Modal Title</span>
        <button class="close-btn">×</button>
      </div>
      <div class="modal-body">
        <p>Modal content goes here. This can be any React component.</p>
      </div>
    </div>
  </div>
</PreviewBox>

```tsx
<Modal
  visible={isOpen}
  onClose={() => setIsOpen(false)}
  title="Modal Title"
>
  <Text>Modal content goes here.</Text>
</Modal>
```

  onClose={handleClose}

## Confirm Modal

Pre-built confirmation dialog.

<PreviewBox class="p-lg">
  <div class="modal-content sm">
    <div class="modal-header centered">
      <span class="modal-title">Delete Reading?</span>
    </div>
    <div class="modal-body centered">
      <p>This action cannot be undone.</p>
    </div>
    <div class="modal-footer">
      <button class="modal-btn secondary">Cancel</button>
      <button class="modal-btn danger">Delete</button>
    </div>
  </div>
</PreviewBox>

```tsx
<ConfirmModal
  visible={showConfirm}
  onClose={() => setShowConfirm(false)}
  onConfirm={handleDelete}
  title="Delete Reading?"
  message="This action cannot be undone."
  confirmText="Delete"
  destructive
/>
```

## API Reference

### Modal Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `visible` | `boolean` | **Required** | Whether visible |
| `onClose` | `() => void` | **Required** | Close handler |
| `title` | `string` | - | Modal title |
| `size` | `'sm' \| 'md' \| 'lg' \| 'full'` | `'md'` | Modal size |
| `closeOnBackdrop` | `boolean` | `true` | Close on backdrop press |
| `showCloseButton` | `boolean` | `true` | Show close button |
| `children` | `ReactNode` | **Required** | Modal content |
| `footer` | `ReactNode` | - | Footer content |
| `style` | `ViewStyle` | - | Custom styles |

### ConfirmModal Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `visible` | `boolean` | **Required** | Whether visible |
| `onClose` | `() => void` | **Required** | Close handler |
| `onConfirm` | `() => void` | **Required** | Confirm handler |
| `title` | `string` | **Required** | Dialog title |
| `message` | `string` | **Required** | Dialog message |
| `cancelText` | `string` | `'Cancel'` | Cancel button text |
| `confirmText` | `string` | `'Confirm'` | Confirm button text |
| `destructive` | `boolean` | `false` | Destructive action |
