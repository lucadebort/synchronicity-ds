Badge is a small label component used to highlight status, categories, or counts. It supports multiple semantic variants and can be filled or outlined.

## Variants

### Default

Neutral badge for general labels.

<PreviewBox class="flex-row gap-lg">
  <span class="sync-badge default">Label</span>
  <span class="sync-badge default outlined">Outlined</span>
</PreviewBox>

```tsx
<Badge>Label</Badge>
<Badge outlined>Outlined</Badge>
```

### Gold

Primary accent badge for highlighting important items.

<PreviewBox class="flex-row gap-lg">
  <span class="sync-badge gold">Featured</span>
  <span class="sync-badge gold outlined">New</span>
</PreviewBox>

```tsx
<Badge variant="gold">Featured</Badge>
<Badge variant="gold" outlined>New</Badge>
```

### Success

Indicates completion or positive status.

<PreviewBox class="flex-row gap-lg">
  <span class="sync-badge success">Complete</span>
  <span class="sync-badge success outlined">Verified</span>
</PreviewBox>

```tsx
<Badge variant="success">Complete</Badge>
<Badge variant="success" outlined>Verified</Badge>
```

### Warning

Indicates attention needed or in-progress state.

<PreviewBox class="flex-row gap-lg">
  <span class="sync-badge warning">Pending</span>
  <span class="sync-badge warning outlined">Review</span>
</PreviewBox>

```tsx
<Badge variant="warning">Pending</Badge>
<Badge variant="warning" outlined>Review</Badge>
```

### Error

Indicates errors or critical states.

<PreviewBox class="flex-row gap-lg">
  <span class="sync-badge error">Failed</span>
  <span class="sync-badge error outlined">Invalid</span>
</PreviewBox>

```tsx
<Badge variant="error">Failed</Badge>
<Badge variant="error" outlined>Invalid</Badge>
```

### Info

Indicates informational states.

<PreviewBox class="flex-row gap-lg">
  <span class="sync-badge info">Info</span>
  <span class="sync-badge info outlined">Beta</span>
</PreviewBox>

```tsx
<Badge variant="info">Info</Badge>
<Badge variant="info" outlined>Beta</Badge>
```

## Sizes

<PreviewBox class="flex-row gap-lg align-end">
  <span class="sync-badge gold sm">Small</span>
  <span class="sync-badge gold md">Medium</span>
  <span class="sync-badge gold lg">Large</span>
</PreviewBox>

| Size | Font | Padding | Usage |
|------|------|---------|-------|
| `sm` | 10px | 6×2 | Compact lists, tables |
| `md` | 12px | 8×3 | Default, most cases |
| `lg` | 14px | 12×4 | Headers, featured items |

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | **Required** | Badge text |
| `variant` | `'default' \| 'success' \| 'warning' \| 'error' \| 'info' \| 'gold'` | `'default'` | Color variant |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Badge size |
| `outlined` | `boolean` | `false` | Use outline style |
| `style` | `ViewStyle` | - | Custom styles |

## Complete Example

```tsx
import { Badge, Card } from '@synchronicity/react-native';
import { View, Text } from 'react-native';

function ReadingCard({ reading }) {
  return (
    <Card variant="elevated">
      <View style={styles.header}>
        <Text style={styles.title}>{reading.hexagram.name}</Text>
        <Badge
          variant={reading.isNew ? 'gold' : 'default'}
          size="sm"
        >
          {reading.isNew ? 'New' : `#${reading.hexagram.number}`}
        </Badge>
      </View>
      <Text style={styles.date}>{reading.date}</Text>
    </Card>
  );
}
```
