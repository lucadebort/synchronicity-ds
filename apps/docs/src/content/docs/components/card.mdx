# Card

Cards are surfaces that group related content and actions. They provide visual separation and hierarchy within layouts.

## Variants

### Default

The standard card with subtle elevation and border.

<div class="component-preview dark">
  <div class="sync-card">
    <div class="sync-card-title">Hexagram 1</div>
    <div class="sync-card-body">The Creative represents the primal power of light, the spirit, and heaven.</div>

```tsx
<Card>
  <Card.Title>Hexagram 1</Card.Title>
  <Card.Body>
    The Creative represents the primal power of light, the spirit, and heaven.
  </Card.Body>
</Card>
```

### Elevated

Cards with stronger shadow for modal-like contexts.

    <div class="sync-card-title">Reading Saved</div>
    <div class="sync-card-body">Your hexagram reading has been saved to your journal.</div>

```tsx
<Card variant="elevated">
  <Card.Title>Reading Saved</Card.Title>
  <Card.Body>Your hexagram reading has been saved to your journal.</Card.Body>
</Card>
```

### Interactive

Cards that respond to touch/click for navigation.

```tsx

  <Card.Title>View Reading →</Card.Title>
  <Card.Body>Tap to see your complete hexagram interpretation.</Card.Body>
</Card>
```

## Anatomy

A card consists of these optional sections:

<div class="component-preview dark">
  <div class="sync-card sync-card-anatomy">
    <div class="sync-card-header">
      <span class="anatomy-label">Header</span>
    </div>
    <div class="sync-card-media">
      <span class="anatomy-label">Media</span>
    </div>
    <div class="sync-card-content">
      <div class="sync-card-title">Title</div>
      <div class="sync-card-body">Body content goes here with supporting text.</div>
    </div>
    <div class="sync-card-actions">
      <span class="anatomy-label">Actions</span>
    </div>
  </div>
</div>

| Section | Purpose |
|---------|---------|
| **Header** | Optional top section for badges, icons, or metadata |
| **Media** | Optional image or visual content area |
| **Content** | Main content area with title and body |
| **Actions** | Optional bottom section for buttons or links |

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'elevated' \| 'interactive'` | `'default'` | Visual style |
| `padding` | `'sm' \| 'md' \| 'lg'` | `'md'` | Internal padding |
| `onPress` | `() => void` | - | Press handler (makes card interactive) |
| `disabled` | `boolean` | `false` | Disable interaction |
| `accessibilityLabel` | `string` | - | Screen reader label |

### Tokens Used

```ts
import { card } from '@synchronicity/tokens/components';

// Background per theme
card.background.light      // #f5f2ea
card.background.dark       // #12121a
card.background.trueBlack  // #0a0a0f

// Layout
card.padding.md            // 16
card.borderRadius          // 12
card.gap                   // 12

// Border
card.border.width          // 1
card.border.color.dark     // #24242f

// Shadow
card.shadow.shadowColor    // #d4af37 (gold-tinted)
card.shadow.shadowOpacity  // 0.08
```

## Usage Guidelines

### Do

- Use cards to group related information
- Keep card content focused on a single topic
- Use consistent padding across similar cards
- Provide clear visual hierarchy within cards

### Don't

- Nest cards within cards
- Use too many interactive cards on one screen
- Mix card sizes inconsistently
- Add too many actions to a single card

<Aside type="tip">
In contemplative applications, cards should feel like quiet containers—not demanding attention but gently holding content. Use elevation sparingly.
</Aside>

---

## Accessibility

- **Focus indicator**: 2px gold outline for interactive cards
- **Touch target**: Entire card surface is tappable
- **Screen reader**: Announce card role and content
- **Contrast**: Card borders meet 3:1 minimum against background

```tsx
<Card
  variant="interactive"
  onPress={handlePress}
  accessibilityLabel="View hexagram reading for The Creative"
  accessibilityRole="button"
>
  <Card.Title>The Creative</Card.Title>
</Card>
```

## Complete Example

```tsx
import { Card } from '@synchronicity/react-native';
import { HexagramVisual } from '../components/HexagramVisual';

function ReadingCard({ reading }) {
  return (
    <Card
      variant="interactive"
      padding="lg"
      onPress={() => navigateToReading(reading.id)}
    >
      <Card.Header>
        <Badge>{reading.date}</Badge>
      </Card.Header>

      <Card.Media>
        <HexagramVisual hexagram={reading.hexagram} size="sm" />
      </Card.Media>

      <Card.Title>{reading.hexagram.name}</Card.Title>
      <Card.Body>{reading.question}</Card.Body>

      <Card.Actions>
        <Button variant="ghost" size="sm">
          Share
        </Button>
      </Card.Actions>
    </Card>
  );
}
```
