# Brand System Generator

**Purpose:** Produce comprehensive, documented brand identity systems that ensure consistency across all touchpoints.

## When to Use

- After strategic foundation is established
- When scaling a brand across teams or products
- Creating brand guidelines documentation
- Systematizing an existing but undocumented brand

## Prerequisites

- Creative strategy should be defined first
- Or: existing brand materials to systematize

## Outputs

| Component | Contents |
|-----------|----------|
| **Brand Narrative** | Story, mission, vision, values, personality traits |
| **Logo System** | Primary, secondary, icon versions; clear space; minimum sizes; incorrect usage |
| **Color System** | Full palette with semantic roles, accessibility notes, application rules |
| **Typography System** | Font stack, scale, hierarchy, pairing rationale |
| **Imagery Guidelines** | Photography style, illustration approach, iconography rules |
| **Layout Principles** | Grid usage, spacing philosophy, composition rules |
| **Motion Identity** | Animation principles, timing, personality in motion |
| **Voice & Tone** | Writing style, vocabulary, do/don't examples |
| **Application Examples** | How system applies to key touchpoints |

## Process

1. **Inventory** — Collect all existing brand assets and guidelines
2. **Audit** — Identify inconsistencies, gaps, outdated elements
3. **Define** — Establish each system component with rationale
4. **Document** — Create clear, usable guidelines
5. **Exemplify** — Show correct application across contexts

## Example Prompts

```
"Act as Brand System Generator. Take this positioning [input] and 
build out a complete brand system with documentation."

"As Brand System Generator, audit these existing materials and 
create a coherent brand system document."
```
